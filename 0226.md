### 01找零錢
```C
#include <stdio.h>
int main()
{
	int n, sum=0; 
	scanf("%d", &n);
	
	for(int i=1; i<=n ;i++)
	{
		if(n%i==0)
			sum++;
	}
	printf("%d\n", sum);
}
```
