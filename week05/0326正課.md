### 01｜字母陣列
```c
#include <stdio.h>
int main()
{
    char line[100] = "decline";
    char line2[100] = {'p', 'r', 'o', 'p', 'e', 'r', '\0'};

    printf("%s\n", line);
    printf("%s\n", line2);
}
```

### 02｜另外一種字串宣告的方法
```c
#include <stdio.h>
int main()
{
    char line[10]="decline";
    printf("%s\n", line);
    char line2[10]={'p', 'r', 'o', 'p', 'e', 'r', '\0'};
    printf("%s\n", line2);

    char line3[]="majority";
    printf("%s\n", line3);
    char line4[]={'m', 'a', 'j', 'o', 'r', 'i', 't', 'y'};
    printf("若忘記加\\0在結尾: ==%s==\n", line4);
}
```
